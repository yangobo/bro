Conceptos nuevos

Viewport: Área visible del sitio web dentro de la ventana del navegador.

La IA como ayuda para crear código: Herramientas como GitHub Copilot o ChatGPT pueden ayudar a generar código, dar sugerencias y acelerar el desarrollo.

Unidades estáticas (medida fija y constante):

  • px (pixeles)
  • cm (centímetros)
  • in (pulgadas)
  • mm (milímetros)
  • pt (puntos)
  • pc (picas)

Unidades relativas (dependen de otro valor):

  • em (tamaño de fuente relativo al elemento padre)
  • rem (tamaño de fuente relativo al elemento raíz)
  • % (porcentaje, se ajusta al tamaño del contenedor)
  • vw (unidad relativa al ancho de la ventana)
  • vh (unidad relativa a la altura de la ventana)
  • ch (relativa al ancho del carácter "0")
  • lh (relativa a la altura de línea del elemento)

@media: Es una característica del diseño web que permite aplicar estilos CSS según las propiedades del dispositivo o navegador, como ancho, alto u orientación de la pantalla. Con ello se crean sitios responsivos que se adaptan a computadoras, tabletas y móviles.

@property: Permite crear variables personalizadas con más control: puedes definir su tipo, un valor inicial y si se heredan o no. A diferencia de las variables normales, estas pueden animarse y usarse en transiciones.

Etiqueta Picture: Sirve para mostrar diferentes imágenes según el ancho de la pantalla.

Cosas que aprendí extra:

  • Display: Define cómo se comporta un elemento en la página, cómo se muestra y cómo interactúa con otros elementos.
  • Object-Fit: Controla cómo se ajusta una imagen o video dentro de su contenedor.
  • Background-attachment: Define si el fondo de un elemento se mueve o se queda fijo cuando haces scroll.
  • Flex-direction: Se usa dentro de un contenedor display: flex y define la dirección en la que se alinean los hijos.
  • Gap: Define el espacio entre elementos en contenedores flex o grid.

Reset: Es un conjunto de reglas CSS que eliminan los estilos predeterminados del navegador para asegurar una apariencia consistente en todos los navegadores.

Java:

Una función es un bloque de código que realiza una tarea específica. Ejemplo:

function saludar() {
    console.log("¡Hola, Mundo!");
}

Funciones que retornan un valor:

function resta(a, b) {
    return a - b;
}

Llamar una función:

resta(7, 3);

Imprimir resultado en consola:

console.log(resta(7, 3));

getElementById: Método de JavaScript para obtener un elemento del DOM por su ID.

addEventListener: Método de JavaScript para registrar un oyente de eventos en un elemento del DOM.

Flexbox: Modelo de diseño en CSS que permite distribuir y alinear elementos dentro de un contenedor de manera flexible.

Se activa con display: flex

Define dos ejes: Eje principal (main axis) y eje secundario (cross axis).

Order: Controla el orden visual en un contenedor flex.

Align-Self: Permite alinear individualmente un elemento a lo largo del eje secundario.

Flex-Grow: Indica cuánto puede crecer un elemento.

Flex-Basis: Define el tamaño base inicial de un elemento.
